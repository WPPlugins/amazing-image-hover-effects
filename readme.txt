== Description ==
Amazing Image Hover Effects is Purely CSS3 based amazing plugin that will allow user to add 30+ beautiful hover effects to images with captions.User can create multiple categories with multiple shortcodes and each category have unlimited images.
It is the fastest and most easiest plugin to set up in just few minutes.It is totally AJAX based and time saver.

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Amazing Image Hover Effects' and install it
3. Go to Dashboard > Amazing Image Hover Effects.Save options
4. Get shortcode using Get Shortcode Button
5. Use shorcode in Post,Page or Custom Post type
6. Now visit that page and see plugin in action
